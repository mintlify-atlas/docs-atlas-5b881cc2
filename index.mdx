    <h2 className="text-2xl font-semibold text-gray-900 dark:text-white mb-3">Quick start</h2>
    <p className="text-base text-gray-600 dark:text-gray-400">Deploy the complete lab environment in minutes</p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Get the lab files and navigate to the directory:
      
      ```bash
      git clone https://github.com/abelperezr/nokia-bng-lab.git
      cd nokia-bng-lab
      ```
    </Step>

    <Step title="Add Nokia licenses">
      Place your Nokia SR-SIM license files in the configs directory:
      
      ```bash
      mkdir -p configs/license
      cp /path/to/SR_SIM_license.txt configs/license/
      ```
      
      <Note>Nokia SR-SIM licenses are required but not included in the repository. You must obtain them from Nokia.</Note>
    </Step>

    <Step title="Deploy the lab">
      Launch the complete lab topology with Containerlab:
      
      ```bash
      sudo containerlab deploy -t lab.yml
      ```
      
      The deployment creates:
      - 2x Nokia BNG routers (SR-7)
      - OLT and ONT devices for fiber access simulation
      - Transport switch (SR Linux)
      - RADIUS server
      - Telemetry stack (gNMIc, Prometheus, Grafana)
    </Step>

    <Step title="Access services">
      Once deployed, access the lab services:
      
      <Accordion title="Service endpoints">
        - **Grafana**: http://localhost:3030 (admin/admin)
        - **Prometheus**: http://localhost:9090
        - **BNG1 SSH**: `ssh admin@localhost -p 56661` (password: lab123)
        - **BNG2 SSH**: `ssh admin@localhost -p 56664` (password: lab123)
      </Accordion>
    </Step>
  </Steps>
</div>
